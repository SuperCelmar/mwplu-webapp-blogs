Dos and Don’ts

Do: Use tokens

Background: var(--color-info)
color: var(--color-black);
padding: var(--space-4);

Don’t: Hardcode styles

Background: #3B82F6
/* Avoid */
color: #111827; /* hardcoded */
padding: 16px;  /* not tokenized */
